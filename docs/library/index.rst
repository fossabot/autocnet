AutoCNet
========
The Automated Control Network (AutoCNet) library is composed of several sub-module focusing on
different aspects of the control network generation process.  AutoCNet currently takes
a structure from motion style approach to generate `ISIS3 <https://isis.astrogeology.usgs.gov>`_ compatible control networks suitable
for bundle adjustment.
