:mod:`control.control` --- Control Networks
===========================================

The :mod:`control.control` module provides the C object.  This is a subclass of a
pandas dataframe that has additional methods to support control network specific operations.
Both the Dataframe and Series that the C object is composed of are valid pandas objects
offering the full range of pandas functionality.
